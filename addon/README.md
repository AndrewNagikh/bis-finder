# BiSFinder - WoW Addon

Аддон для поиска лучших предметов (Best-in-Slot) для классов и специализаций в World of Warcraft.

## Исправленные проблемы

### ✅ **Клик по кнопке класса:**
- Исправлен callback для кнопок классов
- Теперь при клике на класс корректно открывается экран с предметами
- Обновляется заголовок с названием выбранного класса

### ✅ **Скролл для отображения всех специализаций:**
- Добавлен ScrollFrame для контентной области
- Теперь все 26 DPS специализаций отображаются корректно
- Автоматическая высота контента в зависимости от количества элементов
- Скролл появляется когда контент превышает размер окна

### ✅ **Отображение всех специализаций для DPS:**
- Проверено: 26 DPS специализаций из 13 классов
- Warrior: Fury Warrior, Arms Warrior (2)
- Paladin: Retribution Paladin (1)
- Monk: Windwalker Monk (1)
- Demon Hunter: Havoc Demon Hunter (1)
- Druid: Feral Druid, Balance Druid (2)
- Death Knight: Frost Death Knight, Unholy Death Knight (2)
- Hunter: Beast Mastery, Marksmanship, Survival Hunter (3)
- Mage: Arcane, Frost, Fire Mage (3)
- Shaman: Elemental, Enhancement Shaman (2)
- Warlock: Destruction, Demonology, Affliction Warlock (3)
- Rogue: Subtlety, Assassination, Outlaw Rogue (3)
- Evoker: Devastation, Augmentation Evoker (2)
- Priest: Shadow Priest (1)
- Все специализации теперь отображаются как отдельные кнопки

## Новые возможности

### Боковое меню с выбором источников
- **IcyVeins** - основной источник данных (активен)
- **Archon** - планируемый источник (пока неактивен)

### Кнопки ролей
- **Tank** - танк
- **DPS** - урон (13 классов)
- **Healer** - лекарь

### Кнопки классов
- Стилизованные кнопки с иконками классов
- Показ полных названий специализаций
- Цветовая схема для каждого класса
- Минималистичный дизайн

### Новое отображение предметов
- Тип предмета слева (минималистичный серый цвет)
- Уровень предмета под типом
- Иконки предметов справа в ряд
- Каждый тип предмета в отдельной строке
- Названия предметов под иконками

## Структура файлов

- `BiSFinder.lua` - основной файл аддона
- `UIElements.lua` - элементы интерфейса
- `ClassButtons.lua` - модуль кнопок классов (переиспользуемый)
- `ClassSpecMapping.lua` - маппинг классов и специализаций (переиспользуемый)
- `SourceMenu.lua` - боковое меню выбора источников
- `BiSFinderData.lua` - данные предметов

## Использование

1. Загрузите аддон в игру
2. Используйте команды `/bisfinder` или `/bisf` для открытия интерфейса
3. Выберите роль (Tank/DPS/Healer) в боковом меню
4. Выберите класс из списка кнопок (с прокруткой если нужно)
5. Просматривайте предметы в новом формате
